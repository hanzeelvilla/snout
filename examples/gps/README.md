# GPS NEO-6M

## Pin Descriptions and Wiring

| **NEO-6M PIN** | **FUNCTION**  | **ESP32 PIN** |
|----------------|---------------|---------------|
|       VCC      |  Power Supply |      3.3V     |
|       RX       |  UART Receive | TX2 (GPIO 17) |
|       TX       | UART Transmit | RX2 (GPIO 16) |
|       GND      |     Ground    |      GND      |
